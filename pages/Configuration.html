Directly after unpacking sources copy your config.php.dist into config.php and customize:

  • MAIN_PAGE_NAME - default page name that will be presented (can contain spaces / non-ascii letters)
  • MODIFIED_FORMAT - date format for $lastModified variable
  • VIEW_TEMPLATE - select here template for viewing your pages. It's just php file that use some [[Variables]] to present a page
  • PRINT_TEMPLATE - used by print*.php
  • EDIT_TEMPLATE - used by edit.php
